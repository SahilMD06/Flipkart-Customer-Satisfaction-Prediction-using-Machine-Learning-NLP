# Flipkart Customer Satisfaction Prediction using Machine Learning & NLP

## Overview

This project focuses on predicting customer satisfaction (CSAT) scores using Machine Learning and Natural Language Processing (NLP) techniques on Flipkart customer support data. The project includes complete data preprocessing, exploratory data analysis (EDA), feature engineering, NLP/text preprocessing, dimensionality reduction, model building, hyperparameter tuning, and deployment preparation.

---

## Problem Statement

The objective of this project is to analyze customer support interactions and predict customer satisfaction levels based on various factors such as:

* Customer remarks
* Product category
* Handling time
* Response time
* Customer support interactions
* Order-related information

The project helps identify major factors affecting customer satisfaction and supports better business decision-making.
